# Perfil App 

Um aplicativo "Hub" que reúne ferramentas úteis e jogos em uma única interface simples e intuitiva. Desenvolvido em **Java** no Android Studio.

## Funcionalidades

O app conta com quatro módulos principais:

* **Calculadora de IMC:** Informe seu peso e altura para saber sua classificação de saúde.
* **Jogo JoKenPo:** Jogue "Pedra, Papel e Tesoura" contra o computador em uma melhor de 3.
* **Sorteador Mega-Sena:** Gera 6 números aleatórios para apostas, garantindo que não se repitam.
* **Teste de Felicidade:** Um questionário rápido que calcula seu nível de bem-estar baseado na sua rotina.

## Tecnologias Utilizadas

* **Linguagem:** Java.
* **Interface:** XML (Layouts responsivos).
